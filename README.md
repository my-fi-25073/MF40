# MF40 PCB Project

MF40 is a keyboard PCB project created to develop a shared 40% PCB design.

The project is intended to support multiple row-staggered layouts on a common PCB platform.

## Repository Contents

This repository is intended to include:

- KiCad design files required for PCB manufacturing
- Firmware files, including the QMK firmware source code

## Planned Variants

- 12u row-staggered PCB
- 13u row-staggered PCB, 12-column version
- 13u row-staggered PCB, 13-column version

In total, the project is being designed around these three PCB variants.
